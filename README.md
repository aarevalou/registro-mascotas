# Registro de Mascotas

Aplicación web simple para registrar, buscar y eliminar mascotas.

## Objetivos

- Registrar nuevas mascotas con nombre y RUT del dueño/a.
- Listar y buscar mascotas por nombre o RUT.
- Eliminar mascotas registradas.
